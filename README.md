# Projeto-de-IP

Importe o projeto e rode os seguintes comandos no console/terminal, na pasta do projeto :

Verifique se já existe uma pasta de virtual environment. Ela pode aparecer como ".venv", "venv", ou semelhantes.
Caso não haja, crie um virtual environment chamado 'venv':

```shell
python3 -m venv venv
```


Sete o virtual criado ou já existente para ser usado pelo projeto:
Substitua "<nome>" pelo nome do virtual environment. 
Caso tenha sido criado na etapa anterior, substitua por "venv".

```shell
source <nome>/bin/activate
```

Instale o pygame:

```shell
pip install pygame
```

Teste se o pygame tá rodando:

```shell
python3 -m pygame.examples.aliens
```
