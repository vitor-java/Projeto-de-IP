# Projeto-de-IP

Importe o projeto e rode os seguintes comandos no console/terminal, na pasta do projeto :

Cria um virtual environment chamado 'venv':

```shell
python3 -m venv venv
```


Seta o 'venv' para ser usado pelo projeto:

```shell
source venv/bin/activate
```

Instala o pygame:

```shell
pip install pygame
```

Testa se o pygame tá rodando:

```shell
python3 -m pygame.examples.aliens
```
